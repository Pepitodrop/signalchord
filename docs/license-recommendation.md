# License recommendation

Apache License 2.0 is recommended for original SignalChord platform code because it is permissive, includes an express patent grant and is compatible with the intended connector/ecosystem model.

Before publishing, complete a dependency and asset audit. Source adapters may be constrained by publisher terms; raw/source content is not covered by the code license; model weights and datasets retain their own terms; the Neo4j database edition and managed services have separate commercial terms; the Velato language specification/reference materials must be attributed according to their source terms.

Do not add a root `LICENSE` until ownership and contributor policy are confirmed. A future dual-license or enterprise add-on decision should be recorded in an ADR rather than retroactively assumed.
